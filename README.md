# 🚀 ToDoLit API

![Java](https://img.shields.io/badge/Java-17-orange?style=for-the-badge&logo=openjdk)
![Spring Boot](https://img.shields.io/badge/Spring_Boot-3.5.11-brightgreen?style=for-the-badge&logo=springboot)
![Maven](https://img.shields.io/badge/Maven-C71A36?style=for-the-badge&logo=apache-maven&logoColor=white)

Uma API REST completa, segura e robusta desenvolvida para gerenciar tarefas do dia a dia do usuário. O projeto foi construído seguindo as melhores práticas de desenvolvimento back-end em Java.

---

## 🛠️ Tecnologias e Ferramentas

O ecossistema do projeto foi selecionado para garantir performance, segurança e facilidade de manutenção:

*   **Linguagem:** Java (v17)
*   **Framework Principal:** Spring Boot (v3.5.11)
*   **Gerenciador de Dependências:** Apache Maven
*   **Segurança & Autenticação:** Bcrypt
*   **Banco de Dados:** H2 Database Engine (In-Memory)
*   **Produtividade:** Lombok (redução de código boilerplate)

---

## 🎯 Funcionalidades Principais

*   **API RESTful:** Endpoints bem definidos e padronizados utilizando os métodos HTTP corretos (`GET`, `POST`, `PUT`, `DELETE`).
*   **Autenticação & Autorização:** Controle de acesso seguro para proteger os endpoints da aplicação.
*   **Segurança de Dados:** Criptografia de senhas e proteção de informações sensíveis do usuário.
*   **Tratamento de Exceções Global:** Sistema inteligente de captura de erros que retorna respostas limpas e padronizadas para o cliente da API (evitando vazamento de stacktraces).
*   **Deploy:** Hospedagem e disponibilidade do back-end em ambiente de produção no Render.
